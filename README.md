## Laboratory Work #2: Physical Schema Implementation and Database Population

### Project Overview
The goal of this laboratory work was to transform a logical data model into a functioning physical relational database using the MySQL engine. The project focuses on a specialized e-commerce platform schema.

### Key Achievements:
* [cite_start]**Environment Setup:** Configured a local MySQL server using XAMPP and connected via DBeaver[cite: 19, 20].
* [cite_start]**DDL Implementation:** Executed SQL scripts to create tables for Users, Categories, Products, and Orders with proper relational constraints[cite: 22, 23].
* [cite_start]**Data Population:** Developed DML scripts to populate the database with realistic test data.
* **Verification:** Validated database integrity and relationships through SQL queries and visual data management tools.
